Ember.js

A framework for creating ambitious web applications.

Stability without Stagnation

In the fast moving and ever changing frontend world, Ember.js is a stable rock to build on. While being a solid foundation for ambitious projects it values stability and follows a strict no-breaking-changes policy. All of these characteristics make it the perfect choice for projects that are built for the long term.
